# reminder
Basic Events scheduler.

This application basically allows users to add future events and schedule a reminder. You can readily incorporate this application with your daily reminder applications for seamless amd effective alerts.
This application is builtd with free open-source production ready technologies and a large community continually supportying maintainence and libraries updates from time to time.
Libraries used:
  1. FastApi (uvicorn)
  2. GraphQl( Strawberry )
  3. Postgresql
  4. Apollo client

We will use the above libraries to build, setup and run our application.
